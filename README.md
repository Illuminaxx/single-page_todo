# single-page-todo

Todo List sur une seule page, faites en nodeJS et Angular

# Lancement du projet

Lancer 'Mongod' dans un cmd puis lancer 'nodemon server.js' dans un notre cmd

Se rendre sur votre server web, ici pour moi : http://localhost:8080
